Tipos de imágenes

Lossless

  1. Gif - Graphics Interchange Format

    Paleta

    Máximo 256 colores

    Uso

    1. Animaciones simples
    2. Gráficos con colores planos

  2. Png8 - Portable network graphic

    Paleta

    Máximo 256 colores

    Uso

    1. Uso de transparencia
    2. Sin animación
    3. Ideal para iconos
  3. Png24 - Portable network graphic

    Paleta

    Colores ilimitados

    Uso

    1. Similar a PNG-8
    2. Maneja imágenes fijas de más colores y transparencia
  4. SVG - Scalable vector graphics

    Paleta

    Colores ilimitados

    Uso

    1. Gráficos / logotipos para la web
    2. Retina / pantallas de alta resolución
